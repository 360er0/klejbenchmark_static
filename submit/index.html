Submit

The submission should be a zip file containing prediction files for each tasks. Each prediction file should be named test_pred_TASKNAME.tsv and contain a single column of predictions with a header. The order of predictions must match the rows from a test file.

You can download a sample submission file below:

To submit your results, please send an email to piotr.cezary.rybak@gmail.com.

Please include:

  • The name of the submitted model and the link to more information about the model (e.g. a paper, a github repo, a huggingface.co/models page).
  • A descriptive name of the fine-tuning method and the link to code or a description on how the model was fine-tuned on KLEJ tasks.
  • The zip file with predictions.
